# My Linux Dotfiles

Dotfiles, configuration for my Linux Setup

## Install & Setup

```shell
git clone https://github.com/mertssmnoglu/linux-dotfiles.git
```

```shell
cd linux-dotfiles
```

```shell
chmod +x ./setup.sh
```

```shell
sh ./setup.sh
```
